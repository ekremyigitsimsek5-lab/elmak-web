# 🏡 Emlak Web Sitesi

Bu proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit bir emlak tanıtım web sitesidir.

## 🚀 Proje Hakkında
Bu web sitesi, emlak ilanlarını ve gayrimenkul tanıtımlarını modern ve kullanıcı dostu bir arayüz ile göstermek için tasarlanmıştır.

## 💻 Kullanılan Teknolojiler
- HTML5  
- CSS3  
- JavaScript  

## 📱 Özellikler
- Responsive (mobil uyumlu) tasarım  
- Modern ve sade arayüz  
- Kolay kullanılabilir yapı  
- Temel emlak listeleme tasarımı  

## 🎯 Amaç
Bu proje, web geliştirme becerilerimi geliştirmek ve portföy oluşturmak amacıyla yapılmıştır.
